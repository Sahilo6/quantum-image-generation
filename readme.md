Comparative study of classical and quantum generative models for MNIST image generation. We implemented 5 models:

### Classical Models:
- **VAE** - Variational Autoencoder
- **GAN** - Generative Adversarial Network  
- **VQ-VAE** - Vector Quantized VAE

### Quantum Models:
- **Quantum Attention VAE** - VAE with quantum attention mechanism
- **Quantum GAN** - GAN with quantum circuits

### Innovation:
- Novel **quantum attention mechanism** using parameterized quantum circuits (8 qubits, PennyLane)
- **Adaptive routing** system for intelligent model selection
